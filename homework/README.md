# ДЗ2: Линейная и логистическая регрессия на PyTorch
## Задание 1: Модификация моделей (homework_model_modification.py)
Линейная регрессия: Добавлена ручная реализация L1 и L2 регуляризации прямо в цикл обучения. Написан класс EarlyStopping, который прерывает обучение.

Логистическая регрессия: Базовая модель переписана под многоклассовую классификацию с использованием CrossEntropyLoss. Вместо обычной точности  теперь считаются взрослые метрики через sklearn: Precision, Recall, F1-score и ROC-AUC. В конце строится наглядная матрица ошибок (Confusion Matrix).

## Задание 2: Работа с реальными данными (homework_datasets.py)
Написан универсальный класс CustomCSVDataset.


Датасеты: Для регрессии взяты цены на жилье в Калифорнии. Для классификации взял пассажиров Титаника.

## Задание 3: Эксперименты и анализ (homework_experiments.py)
Подбор гиперпараметров: Сравнил скорость сходимости SGD, Adam и RMSprop. Вывод: Adam с правильно подобранным шагом обучения (lr) сходится значительно быстрее и стабильнее базового SGD.

Feature Engineering: Я сгенерировал данные со скрытой нелинейной зависимостью. Базовая линейная регрессия с ней не справилась. Когда были добавлены новые признаки (квадраты значений, их произведения и статистику), MSE упал.
