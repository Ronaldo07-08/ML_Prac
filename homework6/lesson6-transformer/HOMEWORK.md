# Подготовка данных:
Убедитесь, что в папке transformer_basics лежит файл training_text.txt достаточного объема(1-5 Мб).

#Обучение модели:
Откройте файл train_and_chat.py. В блоке if __name__ == '__main__': раскомментируйте функцию train_generator() и закомментируйте chat(). Запустите скрипт:

```bash
python transformer_basics/train_and_chat.py
```
Модель будет сохранена в папку checkpoints/.

# Чат с нейросетью:
После завершения обучения закомментируйте train_generator() и раскомментируйте chat(). Запустите скрипт снова, чтобы начать общение с обученной моделью:

```bash
python transformer_basics/train_and_chat.py
```


### Для выхода из чата напишите quit или exit.

# Результаты

#### Модель, обученная на текстах Артура Конан Дойла, успешно перенимает авторский стиль и детективную лексику ("Holmes", "room", "gentleman", "case"), генерируя логически связные (в рамках ограниченного контекста) диалоги и описания. Использование Beam Search значительно повысило качество и читаемость генерируемого текста.
