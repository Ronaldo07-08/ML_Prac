# Дообучение ResNet-18 - Задание 6

Применена техника Transfer Learning. Загружена предобученная на ImageNet модель ResNet-18.

Train/Val Split: Так как исходный датасет не имел отложенной выборки для валидации, был реализован динамический сплит тренировочной папки с помощью torch.utils.data.random_split.

Dataset Wrapper: Написана специальная обертка для датасета, чтобы применять аугментации только к тренировочной части, оставляя валидационную выборку чистой (только Resize и Normalize).

Заморозка весов  базовых слоев позволила обучить модель всего за несколько эпох с минимальными вычислительными затратами, достигнув высокой точности классификации.

# Как запустить

## Скачивание датасета
Перейдите по ссылке: https://drive.google.com/drive/folders/1BzUQeJgICv2ZDXA9HEWcfiAXBRi4Ib2a?usp=sharing
Скачайте все файлы из папки
Распакуйте архив в папку data/ в корне проекта
## Очистка датасета
 Удалите папки labels и images из каждой подпапки

```bash
 rm -rf data/train/labels
 rm -rf data/train/images
 rm -rf data/val/labels
 rm -rf data/val/images
 rm -rf data/test/labels
 rm -rf data/test/images
```
